CATION - AI Study Buddy for Exam Preparation

CATION is a personalized AI study engine that converts handwritten notes into structured, exam-aligned learning. The system analyzes handwritten notes, extracts key concepts, maps them to the exam syllabus, and generates tailored explanations, quizzes, and study insights.
The current prototype focuses on CAT preparation, with an architecture designed to support additional exam engines such as JEE, NEET, and UPSC.

Key Features
1. Handwritten note analysis using Vision AI
2. Concept extraction and syllabus mapping
3. Personalized study modes (Teach Me, Quiz, Diagnostic, Exam Insights)
4. Adaptive learning with mastery tracking
5. Multilingual explanations using Azure Neural Text-to-Speech
